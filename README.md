# Tic-tac-toe-game
Tic-tac-toe, is a paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row is the winner. This is a working Python Package and can be installed either locally or via pip from the pypi repository.
